# HR Analytics Dashboard (Power BI)

## 📊 Overview
This dashboard provides insights into employee attrition and workforce trends.

## 🔑 Key Metrics
- Total Employees: 1470
- Attrition: 237
- Attrition Rate: 16.12%
- Average Age: 37
- Average Salary: 6.5K

## 📈 Insights
- Highest attrition in Laboratory Technician & Sales roles
- Most attrition in 26–35 age group
- Majority attrition in salary range up to 5K
